locreport
GitHub silverbucket

Lines of code over time, by role.

Point locreport at a public GitHub repository and it counts lines of code at each point in its history, split by the role each file plays — app, tests, config, docs and data. Comments are counted separately; build and vendored files are left out.

A single total can't show this: piling on tests and docs looks the same as growing the app you maintain. Splitting by role keeps them apart.
